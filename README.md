## Getting Started

### Installation

1. Clone the repo

   ```sh
   git@github.com:sevinchek/hello-rails-front-end.git
   ```

### Usage

1. Install node dependencies

   ```sh
   npm install
   ```
   
4. Run the App

   ```sh
   npm start
   ```

5. Access to the website in a browser with the following link to see the random greeting

   ```sh
   localhost:3000/greeting
   ```
